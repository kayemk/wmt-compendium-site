# WMT Compendium (Site)

Static frontend for the WMT Compendium.
It loads data live from the data repo API:

- https://kayemk.github.io/wmt-compendium-data/api/index.json
- https://kayemk.github.io/wmt-compendium-data/api/units.json
- https://kayemk.github.io/wmt-compendium-data/api/warbands.json

Deployed via GitHub Pages (GitHub Actions).
